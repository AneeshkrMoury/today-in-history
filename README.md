## Description

A lightweight web app that shows **historical events, notable births, and important occurrences** for the current day using the **Wikipedia “On This Day” API**.  

Features include:

- **Today’s events** display with images and links to Wikipedia  
- **Custom date search** to explore history for any day  
- **Dark/Light mode toggle** with smooth transitions  
- Footer badge showing **developer contact and version info**

Built with **HTML, CSS, and JavaScript**. Perfect for history enthusiasts and educational purposes.

---

## Usage

1. Open the app in any modern browser.  
2. Browse **Today’s events** or search for a specific date.  
3. Toggle between **Dark/Light mode** using the sun/moon button.  
4. Use the footer badge to **contact the developer** via email.
